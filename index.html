Fable-arch samples

Explore Fable-arch through fun demos!

Fable-arch is a set of tools for building modern web applications inspired by the elm architecture. It is implemented around a set of abstractions which makes it possible to implement custom renderers if there is a need. Fable-arch comes with a HTML Dsl and a renderer built on top of virtual-dom and all the samples here are using those two tools. Hopefully the samples here show you how to get started and gives you some inspiration about how to build your application using Fable-arch.

You can also contribute more examples by sending us a pull request for the samples directory!

Sample applications

  • Hello world - simple application with one input
  • Clock - demo of how to have the app consume events from outside the app using a producer
  • Nesting - how to nest modules/apps together
  • Subscriber demo - sample that shows how to log everyting from the app to the console
  • Sudoku - port of https://github.com/forki/react-sudoku for comparison with fable-react
  • Todo MVC - the famous Todo MVC
  • Routing - route parsing sample
  • Navigation - a navigation sample
  • Devtools - shows what the devtools look like
  • React - a simple counter that uses react for rendering, also with devtools
